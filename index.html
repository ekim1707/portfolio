Personal Projects

Completed:

PeachtreeData.com

PeachtreeData.com is a versitile life-database management site that also contains the capacity to serve in a social-media fashion. Users can upload various things relevant to them to the site and have it all stored and showcased in an organized, interactive display, including but not limited to, favorite quotations, song lyrics, pictures, videos, notes, journal entries, and places. Users also hold the option of sharing anything they want to their live website newsfeed. Building this app mainly required React, but it also used a good amount of Redux due to its many moving parts.

Github

Widowsite

Widowsite is an interactive front-end React.js website regarding the video game Overwatch. It includes a detailed overview of the game and all its heroes, a media player for gameplay videos, and a fully live forum.

Github

InstaID

This application functions as an everyday identification tool, featuring an administrator desktop page, where employees can be added to a company database, as well as a mobile app platform where employees can receive QR codes encoded with their employee information once having been added. These QR codes can then be scanned with any smart phone camera, revealing a pop-up window containing that employee's information to ensure a safe, easy, and instant verification process.

Github

OWRPG

OWRPG is a large-scale, multi-layered RPG game regarding the game Overwatch, which features a 2v2 battles between a hero and an enemy, along with healers for each side. Actions include primary fire, secondary abilities with cooldowns based upon number of turns, and ultimates based upon charge percentage. This application solely utilized Python to be built.
