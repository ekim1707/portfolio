Featured Project: PeachtreeData.com

PeachtreeData.com is a versitile life-database management site that also contains the capacity to serve in a social-media way. Users can upload anything relevant to them to the site in organized fashion, including but not limited to, favorite quotations, song lyrics, pictures, videos, notes, journal entries, and places, and also hold onto the option of sharing any of these to their account connections through the website newsfeed. This app mainly featured React, as well as a good amount of Redux due to its many moving parts.

Widowsite

Widowsite is an interactive front-end React.js website regarding the video game Overwatch. It includes a detailed overview of the game and all its heroes, a media player for videos of gameplay, and a fully live forum.

InstaID

This application functions as an everyday identification tool, featuring an administrator desktop page, where employees can be added to a company database, as well as a mobile app platform where employees can receive QR codes with their verified credentials once having been added.

Github

OWRPG

OWRPG is a large-scale, multi-layered RPG game regarding the game Overwatch, which features a 2v2 battles between a hero and an enemy, along with healers for each side. Actions include primary fire, secondary abilities with cooldowns based upon number of turns, and ultimates based upon charge percentage.

Github

Marvel Weather

Marvel Weather takes two separate third-party API calls from Marvel Developer and OpenWeather and combines them into an interactive, responsive application that relates weather conditions from across the globe to certain Marvel superheroes.
